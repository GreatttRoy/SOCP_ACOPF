4. 约束条件说明

(1) 根节点电压约束

  • 节点 0 为电源节点（平衡节点），电压幅值恒定为 1.0 p.u.

(2a-2b) 节点净注入功率约束

  • 定义节点的有功和无功净注入功率，包含发电机出力、可调负荷削减和固定负荷

(3a-3b) 潮流平衡约束（Branch Flow Model）

  • 节点的净注入功率等于流出功率减去流入功率（考虑线路损耗）
  • ∑(j,k)∈E\sum_{(j,k) \in E}∑(j,k)∈E​：从节点 jjj 流出的所有支路
  • ∑(i,j)∈E\sum_{(i,j) \in E}∑(i,j)∈E​：流入节点 jjj 的所有支路
  • rij⋅lt,i,jr_{ij} \cdot l_{t,i,j}rij​⋅lt,i,j​：支路 (i,j)(i,j)(i,j) 的有功损耗
  • xij⋅lt,i,jx_{ij} \cdot l_{t,i,j}xij​⋅lt,i,j​：支路 (i,j)(i,j)(i,j) 的无功损耗

(4) 电压降落约束（DistFlow Model）

  • 描述沿支路的电压降落关系
  • 考虑了电阻和电抗引起的电压降以及线路损耗的影响

(5) 二阶锥松弛约束

  • 对原始非凸约束 lt,i,j⋅vt,i=Pt,i,j2+Qt,i,j2l_{t,i,j} \cdot v_{t,i} = P_{t,i,j}^2 + Q_{t,i,j}^2lt,i,j​⋅vt,i​=Pt,i,j2​+Qt,i,j2​ 的凸松弛

(6) 发电机出力约束

  • 限制发电机的有功出力在其容量范围内
  • 上下限可随时段 ttt 变化（时变约束）

(7) 可调负荷约束

  • 限制可调负荷的削减量在允许范围内
  • 上下限可随时段 ttt 变化（时变约束）

(8) 电压上下限约束

6. 模型特点

  1. 凸优化问题：通过二阶锥松弛，将非凸的交流潮流问题转化为凸优化问题，可以高效求解并获得全局最优解

  2. 精确性：在辐射状配电网络中，当满足相角恢复条件时，二阶锥松弛可以得到原问题的精确解

  5. 模型依据：基于 Branch Flow Model (BFM) 和二阶锥松弛 (SOCP relaxation) 技术

7. 基于方向割的锥还原方法

7.1 问题背景

在二阶锥交流潮流优化模型中，二阶锥松弛约束 (5) 对原始非凸约束进行了凸松弛：

lt,i,j⋅vt,i=Pt,i,j2+Qt,i,j2⇒∥[2Pt,i,j2Qt,i,jlt,i,j−vt,i]∥2≤lt,i,j+vt,il_{t,i,j} \cdot v_{t,i} = P_{t,i,j}^2 + Q_{t,i,j}^2 \quad \Rightarrow \quad \left\| \begin{bmatrix} 2P_{t,i,j} \\ 2Q_{t,i,j} \\ l_{t,i,j} - v_{t,i} \end{bmatrix} \right\|_2 \leq l_{t,i,j} + v_{t,i}lt,i,j​⋅vt,i​=Pt,i,j2​+Qt,i,j2​⇒​​2Pt,i,j​2Qt,i,j​lt,i,j​−vt,i​​​​2​≤lt,i,j​+vt,i​

松弛的几何意义：

  • 原始约束要求点 (Pt,i,j,Qt,i,j,lt,i,j,vt,i)(P_{t,i,j}, Q_{t,i,j}, l_{t,i,j}, v_{t,i})(Pt,i,j​,Qt,i,j​,lt,i,j​,vt,i​) 必须位于二阶锥的表面（锥面约束）
  • 松弛后的约束允许点位于二阶锥的内部或表面（锥约束）

在辐射状配电网络中，该松弛通常是紧的（tight），即最优解自然满足等式约束。但在某些情况下，松弛解可能落在锥的内部，此时需要进行锥还原，将解投影到锥面上以获得原问题的可行解。

7.2 锥松弛间隙检测

设 SOCP 求解得到的最优解为 (Pt,i,j∗,Qt,i,j∗,lt,i,j∗,vt,i∗)(P^*_{t,i,j}, Q^*_{t,i,j}, l^*_{t,i,j}, v^*_{t,i})(Pt,i,j∗​,Qt,i,j∗​,lt,i,j∗​,vt,i∗​)，定义锥松弛间隙为：

间隙判定：

  • 若 Δt,i,j≈0\Delta_{t,i,j} \approx 0Δt,i,j​≈0（在数值误差范围内），则松弛是紧的，无需还原
  • 若 Δt,i,j>ϵ\Delta_{t,i,j} > \epsilonΔt,i,j​>ϵ（给定容差），则需要进行锥还原

7.3 基于方向的锥还原方法

7.3.1 Cauchy-Schwarz 不等式

对于支路 (i,j)(i,j)(i,j)，记 xij=(Pt,i,j,Qt,i,j)T\mathbf{x}_{ij} = (P_{t,i,j}, Q_{t,i,j})^Txij​=(Pt,i,j​,Qt,i,j​)T，则二阶锥约束为：

对于任意方向向量 d=(dP,dQ)T\mathbf{d} = (d_P, d_Q)^Td=(dP​,dQ​)T，由 Cauchy-Schwarz 不等式可得：

dTxij∥d∥2≤∥xij∥2≤lt,i,j⋅vt,i\frac{\mathbf{d}^T \mathbf{x}_{ij}}{\|\mathbf{d}\|_2} \leq \|\mathbf{x}_{ij}\|_2 \leq \sqrt{l_{t,i,j} \cdot v_{t,i}}∥d∥2​dTxij​​≤∥xij​∥2​≤lt,i,j​⋅vt,i​​

关键要点：

  • 在同一层内，rrr 保持固定，主要通过改变方向 d\mathbf{d}d 来逼近锥面
  • 只有当方向搜索停滞时，才增大 rrr，进入下一层
  • 这种策略充分利用了当前约束强度，避免过早提升 rrr 导致问题过度约束

8.2 方向生成方法对比与实现建议

8.2.1 方法详细说明（正交方向微扰）

几何原理：

在二维空间中，向量 d=(P,Q)T\mathbf{d} = (P, Q)^Td=(P,Q)T 的正交向量为 d⊥=(−Q,P)T\mathbf{d}^{\perp} = (-Q, P)^Td⊥=(−Q,P)T。沿正交方向微扰相当于在当前方向的切线方向移动。

计算步骤：

# 伪代码
def generate_directions_method(P, Q, alpha=0.2):
    """正交方向微扰法生成新方向"""
    # 1. 计算当前方向的范数
    norm_d = sqrt(P**2 + Q**2)

    # 2. 归一化当前方向
    d_hat = (P / norm_d, Q / norm_d)

    # 3. 计算正交方向（逆时针90度旋转）
    d_perp = (-Q / norm_d, P / norm_d)

    # 4. 生成两个新方向（顺时针和逆时针）
    d_new_1 = (d_hat[0] + alpha * d_perp[0],
               d_hat[1] + alpha * d_perp[1])
    d_new_2 = (d_hat[0] - alpha * d_perp[0],
               d_hat[1] - alpha * d_perp[1])

    # 5. 归一化新方向
    norm_1 = sqrt(d_new_1[0]**2 + d_new_1[1]**2)
    norm_2 = sqrt(d_new_2[0]**2 + d_new_2[1]**2)

    d_new_1 = (d_new_1[0] / norm_1, d_new_1[1] / norm_1)
    d_new_2 = (d_new_2[0] / norm_2, d_new_2[1] / norm_2)

    return d_new_1, d_new_2
